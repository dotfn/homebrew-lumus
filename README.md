# Homebrew tap for Lumus Control

Install [Lumus Control](https://github.com/dotfn/lumus-control) via Homebrew.

```bash
brew tap dotfn/lumus
brew install --cask lumus-control
```

To upgrade:

```bash
brew upgrade --cask lumus-control
```

## Note

This app is ad-hoc signed (not notarized by Apple). The cask automatically removes the quarantine attribute after installation to bypass Gatekeeper. You may still see a security dialog on first launch — click "Open Anyway" in System Settings > Privacy & Security if prompted.
